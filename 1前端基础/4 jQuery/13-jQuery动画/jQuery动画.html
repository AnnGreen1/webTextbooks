jQuery动画

前言

jQuery 之所以能后成为JavaScript当前最流行的库之一，动画效果也是其中的一个重要因素。通过jQuery的动画方法，能够轻松地为网页添加视觉效果，给用户一种全新的体验。

隐藏与显示

在jQuery中，隐藏可以通过 hide() 方法来实现；显示则可以通过 show() 来实现，show() 方法和 hide() 方法是 jQuery 中最基本的动画方法。

  • show ：show( ) 方法会将隐藏的元素显示出来，它会 将元素的 display 属性设置为 hide( ) 前的属性值；

  • hide ：hide( ) 方法会记住原来的 display 属性值，并将元素的属性 设置为 display: none，即将元素隐藏。

注意：
display 属性值是保存在 jQuery 的数据缓存中，所以 display 可以方便以后可以恢复到其初始值。假如一个元素的 display 属性值为 inline，那么当它隐藏后再显示时，这个元素将再次显示为 inline 模式。

hide( )

隐藏，函数如下：

hide([duration][,easing][,callback])

参数说明

duration：即设置隐藏效果的速度。接受Number和String类型的值（默认值：400ms）：

1. 毫秒 （ms）
2. "slow" (相当于 200ms)
3. "fast" （相当于 600ms）

easing：设置动画在不同点上的速度，接受String类型的值：

1. "swing" (默认值。在开始和结尾移动较慢慢，在中间移动较快)
2. "linear" （匀速运动）

￼

￼

callback：回调函数。即用来设置 hide() 执行完成后需要执行的操作。

hide( ) 不传参

隐藏元素的最简单方法是不传参直接调用hide( )，如果没有设置参数，则匹配的元素将被立即隐藏，没有动画例如：

<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <title>jQuery 动画</title>
    <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.6.0.min.js"></script>
    <style>
        div {
            background: paleturquoise;
            width: 700px;
            height: 100px;
            margin: 7px;
            float: left;
        }
    </style>
</head>

<body>
    <script>
        $(window).on("load", function() {
            // 添加6个div到body中
            for(var i = 0; i < 6; i++) {
                $("<div>").appendTo(document.body)
            }
            // 给div绑定点击事件
            $("div").on("click", function() {
                // 被点击后隐藏自身
                $(this).hide();
            })
        })
    </script>
</body>

</html>

show( )

show( ) 方法主要用于显示元素，与 hide() 方法用途正好相反，用法相似。需要注意的是：如果选择的元素是可见的，则这个方法将不会改变任何东西；同样的，如果没有参数，匹配的元素将被立即显示，没有动画。

如下：

<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <title>jQuery 动画</title>
    <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.6.0.min.js"></script>
    <style>
        div {
            background: paleturquoise;
            width: 700px;
            height: 100px;
            margin: 7px;
            float: left;
        }

        #btn {
            display: none;
        }
    </style>
</head>

<body>
    <button id="btn">显示元素</button>
    <div id="content"></div>
    <script>
        $(window).on("load", function () {
            // 给content绑定点击事件
            $("#content").on("click", function () {
                // 被点击后隐藏自身
                $(this).hide(300, "swing", function () {
                    $("<p id='msg'>您点击的元素已被隐藏了.</p>").appendTo(document.body);
                    // 显示按钮
                    $("#btn").css("display", "block");
                });
            })

            // 给按钮绑定事件
            $("#btn").on("click", function () {
                $("#content").show(500, "linear", function () {
                    $("#msg").remove();
                    // 隐藏按钮
                    $("#btn").css("display", "none");
                });
            })
        })
    </script>
</body>

</html>

高度变化

在使用 hide( ) 和 show( ) 时大家应该注意到了，元素的宽高是跟随着发生变化，那么如果只想让宽高发生变化可以通过什么方式去实现呢？

slideUp( )

slideUp( )方法将元素 由下到上缩短隐藏（可以简单记忆为：从下往上擦除元素），在没有参数时，持续时间默认为400毫秒。

<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <title>jQuery 动画</title>
    <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.6.0.min.js"></script>
    <style>
        div {
            background: paleturquoise;
            width: 700px;
            height: 100px;
            margin: 7px;
            float: left;
        }

        #btn {
            display: none;
        }
    </style>
</head>

<body>
    <button id="btn">显示元素</button>
    <div id="content"></div>
    <script>
        $(window).on("load", function () {
            // 给content绑定点击事件
            $("#content").on("click", function () {
                // 被点击后隐藏自身
                $(this).slideUp(300, "swing", function () {
                    $("<p id='msg'>您点击的元素已被隐藏了.</p>").appendTo(document.body);
                    // 显示按钮
                    $("#btn").css("display", "block");
                });
            })

            // 给按钮绑定事件
            $("#btn").on("click", function () {
                $("#content").show(500, "linear", function () {
                    $("#msg").remove();
                    // 隐藏按钮
                    $("#btn").css("display", "none");
                });
            })
        })
    </script>
</body>

</html>

slideUp( )与隐藏显示函数一样，都可以接受参数，有对应的这几种情况：slideUp([duration])、slideUp([duration][,easing])、slideUp([duration][,easing][,callback])，那么他们的用法也一致。

slideDown( )

slideDown( )与slideUp( )方法相反，slideDown( )方法 使元素由上到下延伸显示，在没有参数时，持续时间默认为400毫秒。参数和用法也与slideUp( )一致。

注意：
slideDown( )是让隐藏的元素显示，对已经呈现显示状态的对象是没有作用效果的。 如下：

<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <title>jQuery 动画</title>
    <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.6.0.min.js"></script>
    <style>
        div {
            background: paleturquoise;
            width: 700px;
            height: 100px;
            margin: 7px;
            float: left;
        }

        #btn {
            display: none;
        }
    </style>
</head>

<body>
    <button id="btn">显示元素</button>
    <div id="content"></div>
    <script>
        $(window).on("load", function () {
            // 给content绑定点击事件
            $("#content").on("click", function () {
                // 被点击后隐藏自身
                $(this).slideUp(300, "swing", function () {
                    $("<p id='msg'>您点击的元素已被隐藏了.</p>").appendTo(document.body);
                    // 显示按钮
                    $("#btn").css("display", "block");
                });
            })

            // 给按钮绑定事件
            $("#btn").on("click", function () {
                $("#content").slideDown(500, "linear", function () {
                    $("#msg").remove();
                    // 隐藏按钮
                    $("#btn").css("display", "none");
                });
            })
        })
    </script>
</body>

</html>

slideToggle( )

slideToggle( ) 方法可以在 slideDown( ) 与 slideUp( ) 方法之间进行切换。如果元素向下滑动，则 slideToggle( ) 可向上滑动它们；如果元素向上滑动，则 slideToggle( ) 可向下滑动它们。同时，也与其他几个函数一样，都可以接受参数，用法也一致。

<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <title>jQuery 动画</title>
    <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.6.0.min.js"></script>
    <style>
        div {
            background: paleturquoise;
            width: 700px;
            height: 100px;
            margin: 7px;
            display: none;
        }
    </style>
</head>

<body>
    <button id="btn">切换元素</button>
    <div id="content"></div>
    <script>
        $(window).on("load", function () {

            // 给按钮绑定事件
            $("#btn").on("click", function () {
                $("#content").slideToggle(500, "swing");
            })
        })
    </script>
</body>

</html>

淡入淡出

在使用 css 设置样式时，如果想要让元素在页面不可见，常用的办法就是通过设置样式的display: none。
除此之外还可以通过一些类似的办法来达到这个目的，设置元素透明度 opacity：0，可以让元素不可见，透明度的参数是0~1之间的值，通过改变这个值可以让元素有一个透明度的效果。常见的淡入淡出动画 fadeIn() 和 fadeOut() 方法正是基于这样的原理的。

fadeIn( )

fadeIn( ) 用于淡入已隐藏的元素（也就是 用来显示元素），元素透明度由 完全透明变为完全不透明。没有参数时，持续时间默认为400毫秒。可以接受参数，也有对应的这几种情况：fadeIn([duration])、fadeIn([duration][,easing])、fadeIn([duration][,easing][,callback])，使用方式一致。

注意：
此方法是 让隐藏的元素显示出来的，对已经呈现显示状态的对象是没有作用效果的。 如下：

<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <title>jQuery 动画</title>
    <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.6.0.min.js"></script>
    <style>
        div {
            background: paleturquoise;
            width: 700px;
            height: 100px;
            margin: 10px;
            display: none;
        }
    </style>
</head>

<body>
    <button id="btn">显示元素</button>
    <div id="content"></div>
    <script>
        $(window).on("load", function () {
            // 给btn绑定点击事件
            $("#btn").on("click", function () {
                // 显示content
                $("#content").fadeIn(2000);
            })
        })
    </script>
</body>

</html>

fadeOut( )

与 fadeIn( ) 相反，用于淡出可见的元素（隐藏已显示的元素），没有参数时，持续时间默认为400毫秒。可以接受参数，用法类似。

<html>

<head>
    <meta charset="UTF-8">

    </style>

<body>

    <script>
        $(window).on("load", function () {
            // 给按钮绑定事件

fadeTo( )

淡入淡出 fadeIn 与 fadeOut 都是修改元素样式的 opacity 属性，都有个不足的地方，变化的区间要么是0，要么是1。
那么如果要让元素保持动画效果，比如执行 opacity = 0.5 的效果时，要如何处理？
====>

jQuery提供了 fadeTo( ) 方法，此方法允许渐变为给定的不透明度并保持，函数原型：

fadeTo( duration, opacity [, callback ] )

duration：必需的，表示持续时间，持续时间是以毫秒为单位的，数值越大，动画越慢。默认持续400毫秒的；'fast' 和 'slow' 分别代表 200 和 600 毫秒的延时。

opacity：必需的，值为0和1之间的数字，表示元素的不透明度，将淡入淡出效果设置为给定的不透明度，值越小越透明。0 完全透明，1 完全不透明；

callback：可选的，回调函数，动画完成时执行的函数。

<html>

<head>
    <meta charset="UTF-8">
    <title>jQuery 动画</title>
    <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.6.0.min.js"></script>
    <style>
        div {
            background: paleturquoise;
            width: 700px;
            height: 100px;
            margin: 10px;
            display: none;
        }
    </style>
</head>

<body>
    <button id="btn">显示元素</button>
    <div id="content"></div>
    <script>
        $(window).on("load", function () {
            var isHide = true
            // 给btn绑定点击事件
            $("#btn").on("click", function () {
                // 显示或隐藏content
                $("#content").fadeTo(2000, 0.3, function() {
                    console.log("保持0.3的透明度")
                })
            })
        })
    </script>
</body>

</html>

自定义动画

animate( )

animate( ) 方法允许我们通过修改 CSS 样式来让元素从一个状态改变为另一个状态，从而实现动画的效果。
函数原型如下：

animate( properties [, duration ] [, easing ] [, complete ] )

properties ：必须的。用来设置产生动画效果的一个或多个 CSS 属性。

注意：
当与 animate( ) 方法一起使用时，该属性名称必须是驼峰命名规范，如： paddingLeft 代替 padding-left，marginRight 代替 margin-right，依此类推。

duration：即设置隐藏效果的速度。接受Number和String类型的值：

1. 毫秒
2. "slow"
3. "fast"

easing：设置动画在不同点上元素的速度，接受String类型的值：

1. "swing" (在开或者结尾移动较慢慢，在中间移动较快)
2. "linear" （匀速运动）

callback：回调函数。即用来设置hide()执行完成后需要执行的操作。

<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <title>jQuery 动画</title>
    <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.6.0.min.js"></script>
    <style>
        div {
            background: paleturquoise;
            width: 50px;
            height: 50px;
            margin: 50px;
        }
    </style>
</head>

<body>
    <button id="btn">开始动画</button>
    <div id="content"></div>
    <script>
        $(window).on("load", function () {
            // 给btn绑定点击事件
            $("#btn").on("click", function () {
                $("#content").animate({
                    width: "300px"
                }, 500);
                $("#content").animate({
                    height: "300px"
                }, 500);
                $("#content").animate({
                    "border-radius": "150px",
                }, 500, function () {
                    $("#content").css("background-color", "hotpink")
                });
            })
        })
    </script>
</body>

</html>

jQuery 提供了针对动画的队列功能，我们可以编写多个 animate( ) ，jQuery 会创建包含这些方法调用的“内部”队列，然后逐一运行这些 animate 调用。

停止动画

stop( )

此方法用于在动画或效果完成前对它们进行停止。

<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <title>jQuery 动画</title>
    <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.6.0.min.js"></script>
    <style>
        div {
            background: paleturquoise;
            width: 50px;
            height: 50px;
            margin: 50px;
        }
    </style>
</head>

<body>
    <button id="btn">开始动画</button>
    <button id="stop">停止动画</button>
    <div id="content"></div>
    <script>
        $(window).on("load", function () {
            // 给btn绑定点击事件
            $("#btn").on("click", function () {
                $("#content").animate({
                    width: "300px"
                }, 500);
                $("#content").animate({
                    height: "300px"
                }, 500);
                $("#content").animate({
                    "border-radius": "150px",
                }, 500, function () {
                    $("#content").css("backgroundColor", "hotpink")
                });
            })
            $("#stop").on('click', function () {
                // 停止 #content 元素的动画效果
                $("#content").stop()
            })
        })
    </script>
</body>

</html>

stop(stopAll, goToEnd)

stopAll ：可选的，此参数规定是否应该清除动画队列，即把当前元素接下去尚未执行完的动画队列清空。默认是 false，即仅停止活动的动画，允许任何排入队列的动画向后执行。

goToEnd：可选的，参数规定是否立即完成当前动画。默认是 false，因此，默认地，stop( ) 会清除在被选元素上指定的当前动画。可以用于让正在执行的动画直接到达结束时刻的状态，通常用于后一个动画需要基于前一个动画的末状态的情况，可以通过 stop(false,true) 这种方式来让当前动画直接到达末状态。

如果两者结合起来 stop(true,true)，即停止当前动画直接到达当前动画的末状态，并清空动画队列。

Chaining

在jQuery中，我们还可以把动作（方法）链接起来，Chaining 允许我们在一条语句中允许多个 jQuery 方法（在相同的元素上）。这样的话，浏览器就不必多次查找相同的元素，如需链接一个动作，我们可以直接把该动作追加到之前的动作上，如下：

<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <title>jQuery 动画</title>
    <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.6.0.min.js"></script>
    <style>
        div {
            background: paleturquoise;
            width: 50px;
            height: 50px;
            margin: 50px;
        }
    </style>
</head>

<body>
    <button id="btn">开始动画</button>
    <div id="content"></div>
    <script>
        $(window).on("load", function () {
            // 给btn绑定点击事件
            $("#btn").on("click", function () {
                $("#content")
                    .css('background-color','pink')
                    .animate({width:'200px', height:'200px', top:'300px'}, 400)
                    .animate({width:'200px', height:'100px', left:'300px'}, 400)
                    .animate({width:'100px', height:'100px', top:'0px', 'borderRadius': '50px'}, 400)
                    .animate({width:'0px', height:'0px', left:'0px'}, 400)
            })
        })
    </script>
</body>

</html>
