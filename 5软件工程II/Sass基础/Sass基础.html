                                                                • 一、Sass 入门

                                                                Sass, (英文全称：Syntactically Awesome Stylesheets) 是一个最初由 Hampton Catlin 设计并由 Natalie Weizenbaum 开发的层叠样式表语言，是一种 CSS 的预编译语言， 是一种强化 CSS 的辅助性的工具，在 CSS 原本的语法基础上增加了变量 (variables)、嵌套 (nested rules)、混合 (mixins)、导入 (inline imports) 等高级功能，通过这些扩展能够让 CSS 更加优雅、强大。我们通过 Sass 和 Sass 的样式库（比如 Compass）有助于更好地组织管理样式文件，以便高效地开发项目。

                                                                 

                                                                特色功能

                                                                • 完全兼容 CSS3
                                                                • 在 CSS 基础上增加变量、嵌套 (nesting)、混合 (mixins) 等功能
                                                                • 支持通过函数进行颜色值与属性值的运算
                                                                • 提供了控制指令 (control directives)等高级功能
                                                                • 自定义输出格式

                                                                我们为什么要使用 Sass？ CSS 本身语法不够强大，导致重复编写一些代码，不仅无法实现复用，而且代码也不方便维护。 Sass 可以理解为是一个 CSS 的预处理器，可以帮我们减少 CSS 的重复代码，提高开发效率。

                                                                1、语法格式

                                                                Sass 有两种语法格式。

                                                                sass

                                                                Sass ，最早的语法格式，即缩进格式 (Indented Sass)，是一种简化格式。这种语法的特点是 使用 “缩进” 代替 “花括号” 表示属性属于某个选择器，用 “换行” 代替 “分号” 分隔属性，这种格式以 .sass 作为扩展名。

                                                                scss

                                                                SCSS (Sassy CSS)，是 Sass 3 引入新的语法，其语法完全兼容 CSS3，并且继承了 Sass 的强大功能，因此所有 CSS3 语法在 SCSS 中都是通用的，同时又加入 Sass 的特色功能。SCSS 需要使用分号和花括号而不是换行和缩进，SCSS 对空白符号不敏感，其实就和CSS3语法一样， 其后缀名是可以为 .scss 和 .sass 。

                                                                示例：

                                                                // sass
                                                                #root
                                                                width: 30%
                                                                background-color: #faa
                                                                

                                                                // scss
                                                                #root {
                                                                width: 30%;
                                                                background-color: #faa;
                                                                }

                                                                👍🏻👍🏻👍🏻 推荐使用 SCSS

                                                                2.2、Sass 基础语法

                                                                2.2.1、SassScript

                                                                在 CSS 属性的基础上， Sass 提供了一些名为 SassScript 的新功能。 SassScript 可作用于任何属性，允许属性使用变量、算数运算等额外功能。

                                                                2.2.2、嵌套规则

                                                                Sass 允许我们将一套 CSS 样式嵌套进另一套样式中，那么 内层的样式就会将它外层的选择器作为父选择器，例如：

                                                                x
                                                                /* scss */
                                                                #main p {
                                                                  color: #00ff00;
                                                                  width: 97%;
                                                                  .redbox {
                                                                    background-color: #ff0000;
                                                                    color: #000000;
                                                                  }
                                                                }

                                                                将被编译为：

                                                                x
                                                                /* css */
                                                                #main p {
                                                                    color: #00ff00;
                                                                    width: 97%;
                                                                }
                                                                /* 相当于使用了后代选择器 */
                                                                #main p .redbox {
                                                                    background-color: #ff0000;
                                                                    color: #000000;
                                                                }

                                                                嵌套功能够避免重复输入父选择器，而且令复杂的 CSS 结构更易于管理：

                                                                x
                                                                /* scss */
                                                                #main {
                                                                  width: 97%;
                                                                

                                                                  p, div {
                                                                    font-size: 2em;
                                                                    a { font-weight: bold; }
                                                                  }
                                                                

                                                                  pre { font-size: 3em; }
                                                                }

                                                                将被编译为：

                                                                x
                                                                /* css */
                                                                #main {
                                                                    width: 97%;
                                                                }
                                                                #main p, #main div {
                                                                    font-size: 2em;
                                                                }
                                                                #main p a, #main div a {
                                                                    font-weight: bold;
                                                                }
                                                                #main pre {
                                                                    font-size: 3em;
                                                                }

                                                                2.2.3、父选择器

                                                                在嵌套 CSS 规则时，有时也需要直接使用外层的父选择器，例如，当给某个元素设定 hover 样式时，或者当 body 元素有某个 classname 时，可以 用 & 代表嵌套规则外层的父选择器。

                                                                x
                                                                /* scss */
                                                                a {
                                                                  font-weight: bold;
                                                                  text-decoration: none;
                                                                  &:hover { text-decoration: underline; }
                                                                  body.firefox & { font-weight: normal; }
                                                                }

                                                                编译为:

                                                                x
                                                                /* css */
                                                                a {
                                                                    font-weight: bold;
                                                                    text-decoration: none;
                                                                }
                                                                

                                                                a:hover {
                                                                    text-decoration: underline;
                                                                }
                                                                

                                                                body.firefox a {
                                                                    font-weight: normal;
                                                                }

                                                                编译后的 CSS 文件中 & 将被替换成嵌套外层的父选择器，如果含有多层嵌套，父选择器会一层一层向下传递：

                                                                x
                                                                /* scss */
                                                                #main {
                                                                  color: black;
                                                                  a {
                                                                    font-weight: bold;
                                                                    &:hover { color: red; }
                                                                  }
                                                                }

                                                                将编译为:

                                                                x
                                                                /* css */
                                                                #main {
                                                                    color: black;
                                                                }
                                                                

                                                                #main a {
                                                                    font-weight: bold;
                                                                }
                                                                

                                                                #main a:hover {
                                                                    color: red;
                                                                }

                                                                注意： & 必须作为选择器的第一个字符，其后可以跟随后缀生成复合的选择器。

                                                                2.2.4、属性嵌套

                                                                有些 CSS 属性遵循相同的命名空间 (namespace)，比如 font-family，font-size，font-weight 都以 font 作为属性的命名空间。为了便于管理这样的属性，同时也为了避免了重复输入，Sass 允许将属性嵌套在命名空间中，例如：

                                                                x
                                                                /* scss */
                                                                .funky {
                                                                  font: {
                                                                    family: fantasy;
                                                                    size: 30em;
                                                                    weight: bold;
                                                                  }
                                                                }

                                                                将编译为：

                                                                x
                                                                /* css */
                                                                .funky {
                                                                  font-family: fantasy;
                                                                  font-size: 30em;
                                                                  font-weight: bold; 
                                                                }

                                                                命名空间也可以包含自己的属性值，例如：

                                                                x
                                                                /* scss */
                                                                .funky {
                                                                  font: 20px/24px {
                                                                    family: fantasy;
                                                                    weight: bold;
                                                                  }
                                                                }

                                                                将编译为：

                                                                x
                                                                /* css */
                                                                .funky {
                                                                    font: 20px/24px;
                                                                    font-family: fantasy;
                                                                    font-weight: bold;
                                                                }

                                                                2.2.5、注释

                                                                Sass 支持标准的 CSS 多行注释 /* */，以及单行注释 //，需要注意的是 /* */ 会被完整输出到编译后的 CSS 文件中，而 // 不会被输出到 CSS文件中，例如：

                                                                x
                                                                /* scss */
                                                                /* This comment is
                                                                 * several lines long.
                                                                 * since it uses the CSS comment syntax,
                                                                 * it will appear in the CSS output. */
                                                                body { color: black; }
                                                                

                                                                // These comments are only one line long each.
                                                                // They won't appear in the CSS output,
                                                                // since they use the single-line comment syntax.
                                                                a { color: green; }

                                                                编译为：

                                                                x
                                                                /* css */
                                                                /* This comment is
                                                                 * several lines long.
                                                                 * since it uses the CSS comment syntax,
                                                                 * it will appear in the CSS output. */
                                                                body { color: black; }
                                                                  
                                                                a { color: green; }
                                                                注释插值语句

                                                                我们还可以通过插值语句 #{ } 在注释中插入变量: 

                                                                x
                                                                /* scss */
                                                                $version: "1.32.8";
                                                                /* This CSS is generated by SCSS version #{$version}. */

                                                                编译后：

                                                                x
                                                                /* css */
                                                                /* This CSS is generated by SCSS version 1.32.8. */

                                                                2.2.6、变量

                                                                SassScript 最普遍的用法就是变量，变量以美元符号 $ 开头（与PHP中定义变量的方式相同），赋值方法与 CSS 属性的写法一样：

                                                                定义：

                                                                x
                                                                $bgColor: pink;

                                                                使用:

                                                                x
                                                                body {
                                                                    background: $bgColor;
                                                                }
                                                                作用域

                                                                变量支持块级作用域，嵌套规则内定义 的变量只能在嵌套规则内使用，即 局部变量，不在嵌套规则内定义的变量则可在任何地方使用，即全局变量。那么默认情况下，所有定义在任何选择器之外的变量被认为是全局变量。这意味着他们可以在样式表中任意地方被访问。

                                                                将局部变量转换为全局变量可以添加 !global 声明，如下:

                                                                x
                                                                /* scss */
                                                                $myColor: red;
                                                                

                                                                h1 {
                                                                  $myColor: green !global;  // 全局作用域
                                                                  color: $myColor;
                                                                }
                                                                

                                                                p {
                                                                  color: $myColor;
                                                                }

                                                                编译为:

                                                                x
                                                                /* css */
                                                                h1 {
                                                                  color: green;
                                                                }
                                                                

                                                                p {
                                                                  color: green;
                                                                }
                                                                检查变量是否存在

                                                                Sass提供了两个函数，用来测试一个变量是否存在。我们可以使用 variable-exists 或 global-variable-exists 函数来检查我们的局部变量或全局变量是否分别存在。

                                                                如下案例，我们定义了一个变量，变量值是引用 Google Fonts 的一个绝对路径。然后我们通过判断该变量是否能被实例化来决定是否需要 import：

                                                                x
                                                                $google-font: "http://fonts.googleapis.com/css?family=Alegreya";
                                                                

                                                                @if(global-variable-exists(google-font)) {
                                                                    @import url($google-font);
                                                                }

                                                                最结果是:

                                                                x
                                                                @import url("http://fonts.googleapis.com/css?family=Alegreya");

                                                                2.2.7、数据类型

                                                                SassScript 支持 6 种主要的数据类型：

                                                                • 数字：1, 2, 13, 10px
                                                                • 字符串有引号字符串与无引号字符串，"foo", 'bar', baz
                                                                • 颜色，blue, #04a3f9, rgba(255,0,0,0.5)
                                                                • 布尔型，true, false
                                                                • 空值，null
                                                                • 数组 (list)，用空格或逗号作分隔符，1.5em 1em 0 2em, Helvetica, Arial, sans-serif
                                                                • maps, 相当于 JavaScript 的 object，(key1: value1, key2: value2)
                                                                2.2.7.1、字符串（Strings）

                                                                SassScript 支持 CSS 的两种字符串类型：

                                                                1. 有引号字符串 (quoted strings)，如 "Lucida Grande" 'http://sass-lang.com'
                                                                2. 无引号字符串 (unquoted strings)，如 sans-serif bold
                                                                2.2.7.2、数组（Lists）

                                                                数组 (lists) 指 Sass 处理 CSS 中 margin: 10px 15px 0 0 或者 font-face: Helvetica, Arial, sans-serif 这样通过空格或者逗号分隔的一系列的值。不过实际上，独立的值也可以被视为数组 —— 只包含一个值的数组。

                                                                多维数组

                                                                数组中可以包含子数组，比如 1px 2px, 5px 6px 指的是包含 1px 2px与 5px 6px 两个数组的数组。如果内外两层数组使用相同的分隔方式，那就需要用圆括号包裹内层，所以也可以写成 (1px 2px) (5px 6px)。两者间的区别就在于之前的 1px 2px, 5px 6px 使用逗号分割了两个子数组 (comma-separated)，而 (1px 2px) (5px 6px) 则使用空格分割(space-separated)。

                                                                注意： 当数组被编译为 CSS 时，Sass 不会添加任何圆括号（CSS 中没有这种写法），所以 (1px 2px) (5px 6px) 与 1px 2px, 5px 6px 在编译后的 CSS 文件中是完全一样的。

                                                                数组功能

                                                                在 SassScript 中，数组一般主要是为 SassScript 函数服务，有以下常用函数:

                                                                • nth 函数：可以直接访问数组中的某一项；
                                                                • join 函数：可以将多个数组连接在一起；
                                                                • append 函数：可以在数组中添加新值；
                                                                • @each 指令：能够遍历数组中的每一项。
                                                                2.2.7.3、Maps

                                                                Maps即键值对的集合，键被用于定位值（在css中没有对应的概念）。 和 Lists 不同 的是 Maps 必须被圆括号包围，键值对被逗号分割 ，和 Lists 一样， Maps 主要为 SassScript 函数服务，如:

                                                                • map-get 函数：用于查找键值；
                                                                • map-merge 函数：用于map和新加的键值融合；
                                                                • @each 命令： 可添加样式到一个map中的每个键值对。

                                                                Lists 与 Maps Maps 可用于任何 Lists 可用的地方，在 Lists 函数中 Maps 会被自动转换为 Lists ， 如 (key1: value1, key2: value2) 会被 Lists 函数转换为 key1 value1, key2 value2 ，反之则不能。

                                                                2.2.8、运算

                                                                在 Sass 中，所有的数据类型均支持 == 和 ！= 的运算。

                                                                2.2.8.1、数字运算

                                                                数字类型能够支持常规的运算（如：+, -, *, /, %），并且会自动对不同单位间的运算进行转换。

                                                                x
                                                                /* scss */
                                                                p {
                                                                  width: 1in + 8pt;
                                                                }
                                                                /* 
                                                                 * in 英寸，pt 点
                                                                 * 1pt = 1/72 in              
                                                                */

                                                                编译为:

                                                                xxxxxxxxxx
                                                                /* css */
                                                                p {
                                                                  width: 1.111in; 
                                                                }

                                                                数字类型也可以进行关系运算（<, >, <=, >=）

                                                                除法运算

                                                                在 Sass 中，除法运算使用 / 符号，但是 / 在 CSS 中通常也会起到分隔数字的作用，那么我们如何进行区分呢?

                                                                在以下三种情况中， / 将被视为除法运算符号：

                                                                • 如果值或值的一部分，是变量或者函数的返回值
                                                                • 如果值被圆括号包裹
                                                                • 如果值是算数表达式的一部分

                                                                如下：

                                                                x
                                                                /* scss */
                                                                p {
                                                                  font: 10px/8px;             // Plain CSS, no division
                                                                  $width: 1000px;
                                                                  width: $width/2;            // Uses a variable, does division
                                                                  width: round(1.5)/2;        // Uses a function, does division
                                                                  height: (500px/2);          // Uses parentheses, does division
                                                                  margin-left: 5px + 8px/2px; // Uses +, does division
                                                                }

                                                                编译为：

                                                                x
                                                                /* css */
                                                                p {
                                                                  font: 10px/8px;
                                                                  width: 500px;
                                                                  width: 1;
                                                                  height: 250px;
                                                                  margin-left: 9px;
                                                                }

                                                                在以上案例中，我们在使用变量和 / 时会进行除法运算，如果我们需要在使用变量的时候确保 / 不进行除法运算，可以使用插值语法 #{ } 的方式将变量包裹：

                                                                x
                                                                /* scss */
                                                                p {
                                                                  $width: 1000px;
                                                                  width: #{$width}/2;
                                                                }

                                                                编译为:

                                                                x
                                                                /* css */
                                                                p {
                                                                  width: 1000px/2;
                                                                }
                                                                2.2.8.2、字符串运算

                                                                连接符：+

                                                                通过 + 来实现字符串的拼接：

                                                                x
                                                                /* scss */
                                                                p {
                                                                  cursor: e + -resize;
                                                                }

                                                                编译为：

                                                                x
                                                                /* css */
                                                                p {
                                                                  cursor: e-resize;
                                                                }

                                                                注意：

                                                                • 有引号字符串连接无引号字符串，运算结果是有引号的。

                                                                • 无引号字符串连接有引号字符串，运算结果则没有引号。

                                                                  总之，是否有引号取决于第一个字符串

                                                                x
                                                                /* scss */
                                                                p:before {
                                                                content: "Foo " + Bar;
                                                                font-family: sans- + "serif";
                                                                }

                                                                编译为:

                                                                x
                                                                /* css */
                                                                p:before {
                                                                content: "Foo Bar";
                                                                font-family: sans-serif;
                                                                }

                                                                在有引号的文本字符串中如果要插入动态的值或者变量，我们可以通过插值语法 #{ } 来实现：

                                                                x
                                                                /* scss */
                                                                $content: 15;
                                                                

                                                                p:before {
                                                                  content: "I ate #{5 + 10} pies!";
                                                                  content: "I ate #{$content} pies!";
                                                                }

                                                                编译为：

                                                                x
                                                                /* css */
                                                                p:before {
                                                                  content: "I ate 15 pies!";
                                                                  content: "I ate 15 pies!";
                                                                }

                                                                如果我们插入了空值（null），则就直接忽略:

                                                                x
                                                                /* scss */
                                                                $content: null;
                                                                

                                                                p:before {
                                                                  content: "I ate #{$content} pies!";
                                                                }

                                                                编译为:

                                                                x
                                                                /* css */
                                                                p:before {
                                                                  content: "I ate  pies!";
                                                                }

                                                                连接符：空格

                                                                当我们想要使用运算表达式和其他值连用时，需要用空格做连接符，如下案例中，我们想要将运算表达式和 auto 一起使用时，我们需要通过空格进行连接：

                                                                x
                                                                /* scss */
                                                                p {
                                                                  margin: 10px + 10px auto;
                                                                }

                                                                编译为:

                                                                x
                                                                /* css */
                                                                p {
                                                                  margin: 20px auto;
                                                                }

                                                                Tips: 常用的字符串函数如下：

                                                                • quote($string) 函数：给字符串添加引号
                                                                • unquote($string) 函数：删除字符串中的引号。
                                                                • to-upper-case() 函数：将字符串小写字母转换成大写字母。
                                                                • to-lower-case() 函数：函数将字符串大写字母转换成小写字母。

                                                                2.2.9、函数

                                                                SassScript 定义了多种函数，部分函数可以通过普通的 CSS 语句来调用：

                                                                x
                                                                /* scss */
                                                                p {
                                                                  color: rgb(15, 125, 125);  // 颜色函数
                                                                }

                                                                编译为:

                                                                x
                                                                /* css */
                                                                p {
                                                                  color: #0f7d7d;
                                                                }

                                                                 

                                                                二、Sass 高级用法

                                                                Sass 支持所有的 CSS @-Rules，以及 Sass 特有的 “指令”（directives）。

                                                                @-Rules

                                                                @-Rules 是什么？

                                                                一般就是以 @ 开头的一些规则，在 CSS 中有很多 @-Rules 是我们经常会用的，比如 @import、@if、@each、@while、@media、@font-face 等等。在 Sass 中除了支持 CSS 所有的 @-Rules 外，Sass 还提供了一些扩展的 @ 规则。

                                                                @import 导入文件

                                                                Sass 拓展了 @import 的功能，允许其导入 SCSS 或 Sass 文件。被导入的文件将合并编译到同一个 CSS 文件中，另外，被导入的文件中所包含的变量或者混合指令 (mixin) 都可以在导入的文件中使用。

                                                                CSS 中的用法

                                                                在 CSS 中，我们通过 @import 命令用于从其他样式表导入样式规则，如下：

                                                                xxxxxxxxxx
                                                                @import "style.css";

                                                                实际上相当于通过 link 标签来引入样式：

                                                                xxxxxxxxxx
                                                                <link rel="stylesheet" href="style.css" />
                                                                语法格式

                                                                语法格式：@import <url> <media_query_list>

                                                                其中的：

                                                                • <url>：使用绝对或相对地址导入的外部样式表文件。可以是 url( url )函数 或者直接是一个 url字符串
                                                                • <media_query_list>：指定媒体类型和查询条件。

                                                                如下：

                                                                xxxxxxxxxx
                                                                @import "style.css";
                                                                @import url(style.css);
                                                                @import url("style.css");
                                                                

                                                                // 以上3种方式都有效。当使用url(url)的方式时，包住路径的引号可有可无；如果是直接使用url，那么路径的引号必须保留。

                                                                如果再指定媒体查询：

                                                                xxxxxxxxxx
                                                                @import url(style.css) screen and (min-width:800px);
                                                                @import url(style.css) screen and (width:800px),(color);
                                                                @import url(style.css) screen and (min-device-width:500px) and (max-device-width:1024px);

                                                                Tips: 我们需要注意的是，不论是 link 还是 import 方式，都是会先下载所有 css 文件，然后根据媒体去应用 css 样式，而不是根据媒体去选择性下载 css 文件。

                                                                Sass 中的用法

                                                                Sass 拓展了 @import 的功能，允许其导入 SCSS 或 Sass 文件，被导入的文件将合并编译到同一个 CSS 文件中，另外，被导入的文件中所包含的变量或者混合指令 (mixin) 都可以在导入的文件中使用。

                                                                通常，@import 会寻找 Sass 文件并将其导入，但在以下情况下，@import 仅作为普通的 CSS 语句，不会导入任何 Sass 文件：

                                                                • 文件拓展名是 .css
                                                                • 文件名以 http:// 开头；
                                                                • 文件名是 url()；
                                                                • @import 包含 media queries。

                                                                其他情况如扩展名是 .scss 或 .sass，则会导入成功。如果没有指定扩展名，Sass 将会试着寻找文件名相同，扩展名是 .scss 或 .sass 的文件并将其导入。

                                                                导入多个文件

                                                                Sass 允许我们同时导入多个文件，例如同时导入 rounded-corners 与 text-shadow 两个文件：

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                @import "rounded-corners", "text-shadow";
                                                                使用插值语句

                                                                我们导入文件时也可以使用 #{ } 插值语句，不过需要注意的是 只能作用于 CSS 的 url( ) 导入方式，不是通过变量动态导入 Sass 文件:

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                $family: unquote("Droid+Sans"); // 删除字符串中的引号
                                                                @import url("http://fonts.googleapis.com/css?family=#{$family}");

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                @import url("http://fonts.googleapis.com/css?family=Droid+Sans");
                                                                嵌套 @import

                                                                在大多数情况下，一般是在文件的最外层（不在嵌套规则内）使用 @import，不过，我们也可以将 @import 嵌套进 CSS 样式或者 @media 中，与平时的用法效果相同，不过需要注意的是 这样导入的样式只能出现在嵌套的层中， 如下：

                                                                设我们事先定义了 example.scss 文件:

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                .example {
                                                                  color: red;
                                                                }

                                                                将 example.scss 文件导入到 #main 样式中：

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                #main {
                                                                  @import "example";
                                                                }

                                                                那么就会被编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                #main {
                                                                  /* scss */
                                                                }
                                                                #main .example {
                                                                  color: red;
                                                                }

                                                                注意： 不可以在混合指令 (mixin) 或控制指令 (control directives) 中嵌套 @import

                                                                @media 媒体查询

                                                                Sass 中 @media 指令与 CSS 中用法一样，不过在Sass中 允许其在 CSS 规则中嵌套。

                                                                如果 @media 嵌套在 CSS 规则内，编译时，@media 将被编译到文件的最外层，包含嵌套的父选择器。

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                .sidebar {
                                                                  width: 300px;
                                                                  @media screen and (orientation: landscape) {
                                                                    width: 500px;
                                                                  }
                                                                }

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                .sidebar {
                                                                  width: 300px;
                                                                }
                                                                @media screen and (orientation: landscape) {
                                                                  .sidebar {
                                                                    width: 500px;
                                                                  }
                                                                }

                                                                控制指令

                                                                SassScript 给我们提供了一些基础的控制指令，比如在满足一定条件时引用样式，或者设定范围重复输出格式。控制指令是一种高级功能，日常编写过程中并不常用到，一般主要与混合指令 (mixin) 配合使用，尤其是用在 Compass 等样式库中。

                                                                @if 指令

                                                                与常规编程语言中的 if 条件语句类似，当 @if 的表达式返回值不是 false 或者 null 时，则条件成立，那么就输出 { } 内的代码：

                                                                x
                                                                /* scss */
                                                                $width: 50%;
                                                                $height: 80%;
                                                                

                                                                p {
                                                                  @if $width > 60% { border: 3px solid; }
                                                                  @if $height < 90% { border: 4px dotted; }
                                                                  @if null  { border: 1px double; }
                                                                }

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                p {
                                                                  border: 4px dotted;
                                                                }

                                                                @else 与 @else if 指令

                                                                @if 声明后面还可以跟多个 @else if 声明，或者一个 @else 声明。

                                                                与常规的 if else-if 语句用法类似， 如果 @if 声明失败，Sass 将逐条执行 @else if 声明，如果全部失败，最后执行 @else 声明，例如：

                                                                x
                                                                /* scss */
                                                                $type: monster;
                                                                p {
                                                                  @if $type == ocean {
                                                                    color: blue;
                                                                  } @else if $type == matador {
                                                                    color: red;
                                                                  } @else if $type == monster {
                                                                    color: green;
                                                                  } @else {
                                                                    color: black;
                                                                  }
                                                                }

                                                                编译为：

                                                                x
                                                                /* css */
                                                                p {
                                                                  color: green;
                                                                }

                                                                @for 指令

                                                                @for 指令可以在限制的范围内重复输出格式，每次按要求（变量的值）对输出结果做出变动。

                                                                两种格式：

                                                                xxxxxxxxxx
                                                                @for $var from <start> through <end>
                                                                

                                                                @for $var from <start> to <end>

                                                                两种格式的区别在于 through 与 to 的含义：

                                                                • 当使用 through 时，条件范围 包含 start 与 end 的值；
                                                                • 当使用 to 时条件范围只包含 start 的值，不包含 end 的值。

                                                                注意： $var 可以是任何变量，比如 $i； <start> 和 <end> 必须是整数值。

                                                                如下，当使用 through 时：

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                // through
                                                                @for $i from 1 through 3 {
                                                                  .item-#{$i} { width: 2em * $i; }
                                                                }

                                                                编译为:

                                                                xxxxxxxxxx
                                                                /* css */
                                                                .item-1 {
                                                                  width: 4em;
                                                                }
                                                                

                                                                .item-2 {
                                                                  width: 4em;
                                                                }
                                                                

                                                                .item-3 {
                                                                  width: 4em;
                                                                }

                                                                当使用 to 时，如下：

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                // to
                                                                @for $i from 1 to 3 {
                                                                  .item-#{$i} { width: 2em * $i; }
                                                                }

                                                                编译为:

                                                                xxxxxxxxxx
                                                                /* css */
                                                                .item-1 {
                                                                  width: 2em;
                                                                }
                                                                

                                                                .item-2 {
                                                                  width: 4em;
                                                                }

                                                                @each

                                                                @each 的功能与 @for 类似，指令的格式是：

                                                                xxxxxxxxxx
                                                                @each $var in <list>

                                                                $var 可以是任何变量（单个变量或者数组），而 <list> 是连串的值，也就是值列表。

                                                                • 当 $var 为单个变量（如下的 $animal）时：
                                                                xxxxxxxxxx
                                                                @each $animal in puma, jaguar, egret, salamander {
                                                                  .#{$animal}-icon {
                                                                    background-image: url('/images/#{$animal}');
                                                                  }
                                                                }

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                .puma-icon {
                                                                  background-image: url("/images/puma");
                                                                }
                                                                

                                                                .jaguar-icon {
                                                                  background-image: url("/images/jaguar");
                                                                }
                                                                

                                                                .egret-icon {
                                                                  background-image: url("/images/egret");
                                                                }
                                                                

                                                                .salamander-icon {
                                                                  background-image: url("/images/salamander");
                                                                }
                                                                • 当 $var 为数组时，此时 list 是二维数组形式：
                                                                xxxxxxxxxx
                                                                /* scss */
                                                                @each $animal, $color, $cursor in (puma, black, default),
                                                                                                  (jaguar, blue, pointer),
                                                                                                  (egret, white, move) {
                                                                  .#{$animal}-icon {
                                                                    background-image: url('/images/#{$animal}.png');
                                                                    border: 2px solid $color;
                                                                    cursor: $cursor;
                                                                  }
                                                                }

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                .puma-icon {
                                                                  background-image: url("/images/puma.png");
                                                                  border: 2px solid black;
                                                                  cursor: default;
                                                                }
                                                                

                                                                .jaguar-icon {
                                                                  background-image: url("/images/jaguar.png");
                                                                  border: 2px solid blue;
                                                                  cursor: pointer;
                                                                }
                                                                

                                                                .egret-icon {
                                                                  background-image: url("/images/egret.png");
                                                                  border: 2px solid white;
                                                                  cursor: move;
                                                                }
                                                                • 当 <list> 为 maps 类型时：
                                                                xxxxxxxxxx
                                                                /* scss */
                                                                @each $header, $size in (h1: 2em, h2: 1.5em, h3: 1.2em) {
                                                                  .#{$header} {
                                                                    font: {
                                                                      size: $size;
                                                                    }
                                                                  }
                                                                }

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                .h1 {
                                                                  font-size: 2em;
                                                                }
                                                                

                                                                .h2 {
                                                                  font-size: 1.5em;
                                                                }
                                                                

                                                                .h3 {
                                                                  font-size: 1.2em;
                                                                }

                                                                @while

                                                                @while 指令重复输出格式直到表达式返回结果为 false。这样可以实现比 @for 更复杂的循环，如下：

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                $index: 1;
                                                                

                                                                @while $index < 5 {
                                                                  .item-#{$index} {
                                                                    width: 2em * $index;
                                                                  }
                                                                  $index: $index + 1;
                                                                }

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                .item-1 {
                                                                  width: 2em;
                                                                }
                                                                

                                                                .item-2 {
                                                                  width: 4em;
                                                                }
                                                                

                                                                .item-3 {
                                                                  width: 6em;
                                                                }
                                                                

                                                                .item-4 {
                                                                  width: 8em;
                                                                }

                                                                混合指令

                                                                混合指令（Mixin） 用于 定义可重复使用的样式，避免了使用无语意的 class，比如 .float-left。混合指令可以包含所有的 CSS 规则，绝大部分 Sass 规则，甚至通过参数功能引入变量，输出多样化的样式。

                                                                1、定义混合指令 @mixin

                                                                混合指令的用法是在 @mixin 后添加名称与样式，如下我们定义了名称为 large-text 的混合指令：

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                @mixin large-text {
                                                                  font: {
                                                                    family: Arial;
                                                                    size: 20px;
                                                                    weight: bold;
                                                                  }
                                                                  color: #ff0000;
                                                                }

                                                                混合也需要包含选择器和属性，甚至可以用 & 引用父选择器：

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                @mixin clearfix {
                                                                  display: inline-block;
                                                                  &:after {
                                                                    content: ".";
                                                                    display: block;
                                                                    height: 0;
                                                                    clear: both;
                                                                    visibility: hidden;
                                                                  }
                                                                  * html & { height: 1px }
                                                                }

                                                                2、引用混合样式 @include

                                                                我们可以通过 @include 指令来引用混合样式，格式是在其后添加混合指令的名称，以及需要的参数（可选）：

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                @mixin large-text {
                                                                  font: {
                                                                    family: Arial;
                                                                    size: 20px;
                                                                    weight: bold;
                                                                  }
                                                                  color: #ff0000;
                                                                }
                                                                

                                                                .page-title {
                                                                  @include large-text;
                                                                  padding: 4px;
                                                                  margin-top: 10px;
                                                                }

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                .page-title {
                                                                  font-family: Arial;
                                                                  font-size: 20px;
                                                                  font-weight: bold;
                                                                  color: #ff0000;
                                                                  padding: 4px;
                                                                  margin-top: 10px;
                                                                }

                                                                混合样式中还可以包含其他样式，如下:

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                @mixin highlighted-background {
                                                                  background-color: #fc0;
                                                                }
                                                                @mixin header-text {
                                                                  font-size: 20px;
                                                                }
                                                                

                                                                @mixin compound {
                                                                  @include highlighted-background;
                                                                  @include header-text;
                                                                }
                                                                

                                                                .page-title {
                                                                  @include compound;
                                                                  padding: 4px;
                                                                  margin-top: 10px;
                                                                }

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                .page-title {
                                                                  background-color: #fc0;
                                                                  font-size: 20px;
                                                                  padding: 4px;
                                                                  margin-top: 10px;
                                                                }

                                                                Tips： 混合样式中应该 只定义后代选择器，这样可以安全的导入到文件的任何位置。

                                                                参数

                                                                参数用于给混合指令中的样式设定变量，并且赋值使用(与函数的形式参数类似)。

                                                                在定义混合指令的时候，在指令名称后面通过小括号来添加参数，多个参数之间需要用逗号进行分隔(用法与定义函数参数类似)。引用指令时，按照参数的顺序，再将所赋的值对应写进括号（相当于函数调用时传递的实际参数）：

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                @mixin sexy-border($color, $width, $style) {
                                                                  border: {
                                                                    color: $color;
                                                                    width: $width;
                                                                    style: $style;
                                                                  }
                                                                }
                                                                

                                                                p {
                                                                  @include sexy-border(blue, 10px, dashed);
                                                                }

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                p {
                                                                  border-color: blue;
                                                                  border-width: 10px;
                                                                  border-style: dashed;
                                                                }

                                                                Tips： 关键词参数 以上的混合指令引用时，我们在传入参数时可以添加关键词参数：

                                                                xxxxxxxxxx
                                                                p {
                                                                @include sexy-border($color:blue, $width: 1in, $style: dashed);
                                                                }

                                                                参数默认值

                                                                与函数参数默认值一样，我们还可以给混合指令的参数设置默认值，只需要在参数的后面添加上默认值即可，如下:

                                                                xxxxxxxxxx
                                                                /* scss */
                                                                @mixin sexy-border($color, $width, $style: solid) {
                                                                  border: {
                                                                    color: $color;
                                                                    width: $width;
                                                                    style: $style;
                                                                  }
                                                                }
                                                                

                                                                p {
                                                                  @include sexy-border(blue, 10px, dashed);
                                                                  div & {
                                                                    @include sexy-border(blue, 10px);
                                                                  }
                                                                }

                                                                编译为：

                                                                xxxxxxxxxx
                                                                /* css */
                                                                p {
                                                                  border-color: blue;
                                                                  border-width: 10px;
                                                                  border-style: dashed;
                                                                }
                                                                div p {
                                                                  border-color: blue;
                                                                  border-width: 10px;
                                                                  border-style: solid;
                                                                }

                                                                自定义函数

                                                                Sass允许用户自定义函数，我们通过 @function 指令来编写自己的函数：

                                                                x
                                                                /* scss */
                                                                @function get-double($num) {
                                                                  @return $num * 2;
                                                                }
                                                                

                                                                #sider {
                                                                  width: get-double(5px);
                                                                }

                                                                编译为:

                                                                xxxxxxxxxx
                                                                /* css */
                                                                #sider {
                                                                  width: 10px;
                                                                }

                                                                Tips： 建议在自定义函数前添加前缀来避免命名冲突，其他人阅读代码时也会知道这不是 Sass 或者 CSS 的自带功能。

                                                                与混合指令中的参数类似的，函数参数也可以设置参数默认值和关键词参数；

                                                                xxxxxxxxxx
                                                                /* scss */
