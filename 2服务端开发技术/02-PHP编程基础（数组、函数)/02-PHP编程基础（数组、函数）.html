var_dump()

有时候，我们除了想要查看某个变量的内容，可能还需要查看变量的类型等一些结构化的信息，那么此时可以使用 var_dump() 函数来进行输出。

var_dump () 函数可以输出变量的相关信息，会显示变量的结构化信息，包括其数据类型、值、长度，元素的个数，它是用于调试代码的，没有返回值。

示例1：

<?php
    $a = array(1, 2, array("a", "b", "c"));
    var_dump($a);
?>

输出的结果是：

array(3) { [0]=> int(1) [1]=> int(2) [2]=> array(3) { [0]=> string(1) "a" [1]=> string(1) "b" [2]=> string(1) "c" } }

示例2：

<?php
    $b = 3.1;
    $c = true;
    var_dump($b, $c);
?>

输出的结果是：

float(3.1) bool(true)

预定义变量（超全局变量）

预定义变量即系统自己定义好的变量，也就是系统内置的一些变量，我们可以直接使用。

在php中，我们自定义的变量的作用域一般分两种：全局作用域（函数外），局部作用域（函数内）。
不过预定义变量的作用域称为 “超全局作用域” ：全局作用域 + 局部作用域，即 函数内外全都可用 。因此，预定义变量也称为：超全局变量

$_POST 用于接收前台表单使用post方式提交的数据
$_GET 用于接收前台表单使用get方式提交的数据
$_FILES 记录用户上传的文件信息
$_ENV 一个包含服务器端环境变量的数组
$_COOKIE 一种会话技术
$_SESSION 一种会话技术

string

字符串是使用单引号或者双引号括起来的0个或多个字符。

单双引号区别

  • 在字符串中，双引号可以识别变量，单引号不识别变量

  • 在单引号字符串中的变量和特殊字符的转义序列将不会被替换。

  • 用双引号定义的字符串最重要的特征是变量会被解析。

$a = 1 ;
$b = ' 123  $a ' ;
$c = " 123 $a " ;

//得到输出结果：
123 $a
123 1

转换成字符串

一个值可以通过在其前面 拼接上 string 的值 或用 strval() 函数 来转化成字符串。在一个需要字符串的表达式中，会自动转换为 string。

注意：

null 总是被转变成空字符串。

<?php
  $val1 = "" . 3;
  var_dump($val1); // 结果：string(1) "3"
  $val2 = strval(null);
  var_dump($val2); // 结果: string(0) ""
?>

<<<：定义字符串，主要应用场景是定义大段的文本。

注意：
结束时所引用的标识符必须在该行的第一列，而且，标识符的命名也要像其它标签一样遵守 PHP 的规则：只能包含字母、数字和下划线，并且必须以字母和下划线作为开头。

语法格式：

<<<标识符

内容

标识符;

如下：

<?php
  $bar = <<<EOT

  • 开始标记和结束标记可以随意命名但是必须要一致

数组

在 PHP 中，有三种类型的数组：

  1. 索引数组 = 数组元素的下标是数值，这样的数组就是索引数组
  2. 关联数组 = 数组元素的下标是字符串，这样的数组就是关联数组
  3. 多维数组 = 包含一个或多个数组的数组，数组嵌套数组

###语法

定义数组：

  1. 自动分配 ID 键（索引数组）

    $array = array("bar","foo");

// 自 PHP 5.4 起
$array = ["bar","foo"];

// 用 print_r() 调试输出
// Array ( [0] => bar [1] => foo )

  2. 人工分配 ID 键（关联数组）

    $array = array(
    "foo" => "bar",
    "bar" => "foo",
);

// 自 PHP 5.4 起
$array = [
    "foo" => "bar",
    "bar" => "foo",
];

// 用 print_r() 调试输出
// Array ( [foo] => bar [bar] => foo )

  3. 键值对 key=>value

    • 数组中可以保存多个数据 (value，即：值)

    • 每个数据有唯一的编号 (key，即：键)

    • 每个键值对用逗号 , 分割

    # 定义数组，a，b，c 为数据的 key
$arr = ['a' => 1, 'b' => 2 , 'c' => 3] ;
print_r($arr);
# 也可以不写 key，由 php 自动生成，从 0 开始编号
$arr = [ 1, 2, 3 ] ;
print_r($arr);

思考：

$array[1] = "bar";
$array[3] = "foo";

echo $array[2];  // 会输出什么呢？

多维数组

和其他编程语言一样，PHP 中的数组也可以嵌套，即多维数组：

索引数组嵌套索引数组：

<?php
$array = [
  [1, 2, 3],
  ['a', 'b', 'c']
];
echo $array[1][2]; # 输出：c
?>

关联数组嵌套关联数组：

<?php
$array = [
  '第一排' =>
  [
    'a' => '张三',
    'b' => '李四'
  ],
  '第二排' =>
  [
    'c' => '王五',
    'd' => '赵六'
  ]
];
echo $array['第一排']['a']; # 输出：张三
?>

当然，索引数组和关联数组之间也可以互相嵌套:

<?php
$array = [
  '第一排' =>
  [
  '张三',
  '李四'
  ],
  '第二排' =>
  [
    'c' => '王五',
    'd' => '赵六'
  ]

echo $array['第一排'][0]; # 输出：张三
?>

$arr = ['a' => 1, 'b' => 2 , 'c' => 3] ;

// 增
$arr['d'] = 4 ;

// 改
// 改跟 增 一样，如果这个键存在就是修改，不存在就是增加
$arr['d'] = 4 ;

// 删
unset ($arr['c']) ;

// 查
echo $arr['a']

数组的指针

  • current($arr) 用于获取 当前 指针所指向的 元素的值

  • key($arr) 用于获取 当前 指针所指向的 元素的键名

  • next($arr) 用于将数组的 指针下移

  • prev($arr) 用于将数组的 指针上移

  • reset($arr) 用于将数组的指针 重置 (归位，数组的指针默认位于 第1个元素 )

  • end($arr) 用于将数组的指针移到 最后一个元素

<?php
$arr = ['id'=>1,'name'=>'鼠标','price'=>123];

echo key($arr).'=>'.current($arr); // id=>1
next($arr);
echo key($arr).'=>'.current($arr); // name=>鼠标
?>

数组常用操作方法

  • count(数组) 用于获取数组的长度

  • array_keys() 获取数组元素所有的键名

  • array_values() 获取数组元素所有的值

    <?php
$arr = ['id'=>1,'name'=>'鼠标','price'=>123];

echo count($arr);// 3
print_r(array_keys($arr)); // Array ( [0] => id [1] => name [2] => price ) 
print_r(array_values($arr)); // Array ( [0] => 1 [1] => 鼠标 [2] => 123 )

  • array_key_exists(key，arr) 用于判断某个键名是否存在与数组中，如果存在返回true,否则返回false；

  • in_array(value,arr) 用于判断某个键值是否存在与数组中，如果存在返回true，否则返回false；

  • array_merge(数组1，数组2，....) 用于合并数组

  • sort() 对数组按键值进行升序排序，但原下标不会变

  • rsort() 对数组按键值进行降序排序，但原下标不会变

函数

函数是指一段在一起的、可以做某一件事的程序，同一个函数可以被调用任意次。
在程序设计中，常将一些常用的功能模块编写成函数，放在函数库中供公共选用。要善于利用函数，减少重复编写程序段的工作量。

回顾：在 JavaScript 中是如何定义函数的？

语法：

// 创建函数
function functionName(形参列表)
{
    // 函数体
    [return 值]
}

// 调用函数
functionName();

命名规范

  • php的函数名是不区分大小写的，但是建议书写时区分，小驼峰命名
  • 函数的名称应该见名知意
  • 函数名称以字母或下划线开头（不能以数字开头）

示例：

<?php
   function say(){
       echo 'Hi~';
   }
   say();
?>

如果变量 $fun 的值是刚好是函数的名字，那么我们可以通过 $fun() 这种方式来调用该函数：

<?php
   function say(){
       echo 'Hi~';
   }

   $fun = 'say'; # 它的值刚好与上面 say() 函数的名字相同
   $fun();
?>

当然对于普通变量也适用:

<?php
   $say = "Hello";

   $fun = 'say'; # 它的值刚好是上面 $say 变量名
   echo $$fun;
?>

函数参数

**形参：**函数定义时的参数就是形参。

**实参：**函数调用你时的参数就是实参。

示例：

<?php
   function say($content){
       echo 'Hi~'.$content;
   }

   say("你好");
?>
    
    // 形参是被动的接收数据，只对内部有效，对于外部是不可见的，而实参是主动向函数内部传递实际的数据

**形参默认值：**设置形参的时候可以给参数一个默认值，这样如果传递了该参数则使用传递的参数，如果没有传递则使用默认值

示例：

<?php
   $fn = function($content='Hi~'){
       echo $content;
   };
   $fn();
   $fn("你好");
?>

注意：

由于参数传递是按顺序匹配，所以 默认参数一般是最后一个参数

形参的引用传值：

php中函数的实参与形参之间也是一种赋值方式，默认是值传递，在PHP中可以在形参前面加上 & 符号，将实参与形参之间默认的赋值传值更改为引用传值。

我们来看一个例子，交换两个变量的值：

<?php
// 交换两个变量的值
function swapValue($a, $b){
  $tmp = $a;
  $a = $b;
  $b = $tmp;
}

$x = 1;
$y = 2;
swapValue($x, $y);
echo "x：" . $x . "，y：" . $y;
?>

思考：以上代码能否成功？

作如下的修改即可：

<?php
// 交换两个变量的值
function swapValue(&$a, &$b){
  $tmp = $a;
  $a = $b;
  $b = $tmp;
}

$x = 1;
$y = 2;
swapValue($x, $y);
echo "x：" . $x . "，y：" . $y;
?>

匿名函数

没有名字的函数就是匿名函数，也称为闭包函数。

<?php
   function(){
       echo '匿名函数';
   };      // 如果是匿名函数，则结尾一定要有分号;
?>

作用：

  • JS中匿名函数可以自调用(立即执行函数)，但是PHP中的函数是没有办法自调用的
  • PHP中的匿名函数，主要作用是赋值给一个变量或者用于某个函数的参数(回调函数，callback)

示例：

<?php
  $fun = function(){
    echo '匿名函数';
  };      // 如果是匿名函数，则结尾一定要有分号;
  $fun();
?>

PHP常用函数

empty()

**简介：**empty () 函数用于检查一个变量是否为空，当一个变量并不存在，或者它的值等同于 FALSE ，那么它会被认为不存在，如果变量不存在，empty () 并不会产生警告。

语法：empty( $var );

**返回值：**作为一个检查变量是否为空的函数。

  • 当变量为空时，返回 TRUE
  • 当变量不为空时，返回 FALSE

注意：

  • 当字符串的值为 “0” 时，也会认为是空，返回 TRUE
  • 变量为空时是真，不为空时是假

被 empty () 函数 认为是空的值：

  • "" (空字符串)
  • 0 (作为整数的 0)
  • 0.0 (作为浮点数的 0)
  • “0” (作为字符串的 0)
  • NULL
  • FALSE
  • array () (一个空数组)
  • $var; (一个声明了，但是没有赋值的变量)

Session( )

**简介：**session 变量用于存储关于用户会话（session）的信息，或者更改用户会话（session）的设置。Session 变量存储单一用户的信息，并且对于应用程序中的所有页面都是可用的。

在计算机上操作某个应用程序时，打开它，做些更改，然后关闭它。这很像一次对话（Session）。

计算机不知道你是谁。它清楚你在何时打开和关闭应用程序。然而，在因特网上问题出现了：由于 HTTP 地址无法保持状态，Web 服务器并不知道你是谁以及你做了什么。

PHP session 解决了这个问题，它通过在服务器上存储用户信息以便随后使用（比如用户名称、购买商品等）。然而，会话信息是临时的，在用户离开网站后将被删除。如果需要永久存储信息，可以把数据存储在数据库中。

Session 的工作机制是：为每个访客创建一个唯一的 id (UID)，并基于这个 UID 来存储变量。UID 存储在 cookie 中，或者通过 URL 进行传导。

语法：

1、开始 Session

在您把用户信息存储到 PHP session 中之前，首先必须启动会话。

**注意：**session_start () 函数必须位于 <html> 标签之前；

<?php session_start(); ?>

<html>
</html>

2、存储 session 变量

存储和取回 session 变量的正确方法是使用 PHP $_SESSION 变量

//存储
<?php
    session_start();
    // 存储 session 数据
    $_SESSION['views']=1;
?>

//输出
<?php  echo $_SESSION['views']; ?>

3、销毁 Session

删除 session 数据，可以使用 unset () 或 session_destroy () 函数。unset () 函数用于释放指定的 session 变量。

//使用 unset()
unset($_SESSION['views']);

//使用 session()
session_destroy();

header()

**简介：**header函数用于发送原生 HTTP 头。

注意：

header() 必须在任何实际输出之前调用，不管是普通的 HTML 标签，还是文件或 PHP 输出的空行，空格。

示例：

// 1. 跳转页面
header('Location:http://www.baidu.com');

// 2. 在某个时候后执行跳转,单位:秒
header( 'refresh:2; url=http://www.baidu.com' ) ;

file_exists()

简介：file_exists () 函数检查文件或目录是否存在。如果指定的文件或目录存在则返回 TRUE，否则返回 FALSE。

语法：file_exists( "检查的路径" )；

数值相关

取整函数主要有三种：ceil ()、floor ()、round ()

  • ceil() ：进一法取整，向上取整，返回值是float类型

  • floor() ：舍去法取整，向下取整，返回值是float类型

时间相关

date()

格式化时间信息

语法：date(format [, time])

说明：

  • format用于格式化时间的字符串
  • time表示所需要格式化的时间戳，如果省略表示对当前时间进行格式化

示例：

<?php
echo date('Y-m-d H:i:s');   // 2021-09-26 05:21:16

Tips：

如果发现PHP格式化日期显示出来的跟当前时间不一致，那么需要修改时区信息：修改php.ini里的date.timezone = Asia/Shanghai;

time()

获取当前时间的时间戳，单位是秒。时间戳就是从时间原点【1970年1月1日0时0分0秒】至现在的一个秒数。

示例：

<?php
echo time();   // 1632626071
microtime()

获取毫秒数

<?php
echo microtime();   //0.48144900 1632626184
